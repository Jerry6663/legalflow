# LegalFlow 法律知识库

## 概述

LegalFlow 法律知识库是 AI 合同审查产品的核心知识引擎，为合同智能审查提供法律依据、审查规则和合同模板支撑。

## 目录结构

```
knowledge_base/
├── README.md              # 本文件 - 知识库说明
├── contracts/             # 合同类型模板与清单
│   ├── README.md          # 涵盖的合同类型说明
│   ├── 买卖合同模板.md      # 买卖合同标准模板
│   ├── 劳务合同模板.md      # 劳务合同标准模板
│   └── 保密协议模板.md      # 保密协议标准模板
├── laws/                  # 法律法规要点摘录
│   ├── 民法典合同编要点.md   # 民法典合同编核心法条
│   ├── 劳动法要点.md        # 劳动法核心法条
│   └── 公司法要点.md        # 公司法核心法条
└── rules/                 # 合同审查规则库
    └── 合同审查规则.md      # 150+ 条审查规则
```

## 使用方式

1. **合同模板**：作为对照基准，AI 将上传的合同与标准模板进行比对，识别缺失或异常条款。
2. **法律依据**：为审查规则提供法条支撑，AI 据此判断条款的合法性与合规性。
3. **审查规则**：系统将逐条规则应用于合同文本，输出风险提示与修改建议。

## 维护说明

- 合同模板需定期更新以反映最新法律实践
- 法律法规要点需在立法更新后及时修订
- 审查规则库为持续积累型知识资产，建议每季度评审更新
