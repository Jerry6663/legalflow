# 合同类型清单

本知识库目前覆盖以下合同类型的审查与模板支持：

## 已覆盖的合同类型

| 序号 | 合同类型 | 模板文件 | 审查规则覆盖 |
|------|---------|---------|-------------|
| 1 | 买卖合同 | 买卖合同模板.md | 全量覆盖 |
| 2 | 劳务合同 | 劳务合同模板.md | 全量覆盖 |
| 3 | 保密协议（NDA） | 保密协议模板.md | 全量覆盖 |
| 4 | 技术开发合同 | 通用规则覆盖 | 部分覆盖 |
| 5 | 服务合同 | 通用规则覆盖 | 部分覆盖 |
| 6 | 租赁合同 | 通用规则覆盖 | 部分覆盖 |
| 7 | 借款合同 | 通用规则覆盖 | 部分覆盖 |
| 8 | 委托合同 | 通用规则覆盖 | 部分覆盖 |
| 9 | 合伙协议 | 通用规则覆盖 | 部分覆盖 |
| 10 | 担保合同 | 通用规则覆盖 | 部分覆盖 |

## 计划扩展的合同类型

- 建设工程合同
- 运输合同
- 仓储合同
- 保险合同
- 知识产权许可合同
- 投资协议
- 股权转让协议
- 特许经营合同

## 说明

- "全量覆盖"表示该类型有独立模板且审查规则库包含针对性的审查要点
- "部分覆盖"表示适用通用审查规则，暂无专用模板
- 审查规则中的"适用合同类型"字段标识了每条规则的适用范围
